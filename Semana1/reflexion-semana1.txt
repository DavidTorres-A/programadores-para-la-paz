Respuesta:La relación entre estos temas se basa principalmente en la responsabilidad y el orden.Por una parte como ciudadanos digitales, debemos actuar con respeto y ética en internet,pero también aprender a no creer todo lo que vemos las redes sociales siempre continene información falsa,nada verídica por lo tanto antes de compartir información,es importante que se verifique si realmente es verdadera que esta información tenga fuentes que lo comprueben,porque la mayoría de veces se difunden datos falsos que terminan afectando a las personas causando incertidumbre,duda hasta miedo obviamente depende del tipo de información que se comparte.Y esto se conecta con el pensamiento algorítmico que se maneja en herramientas como Git debido a que en un proyecto técnico no es suficiente con subir los archivos sin tener una organización previa o bueno no subir archivos mejor dicho crear las líneas de código con orden sino se tiene un orden previo,si no se trabaja paso a paso el código nunca funcionará,por lo tanto es necesario seguir un proceso como mencionaba anteriormente trabajar paso a paso teniendo el control de todos los cambios que realizamos,ya que pues cada acción que realicemos puede afectar a todo el equipo en este caso si trabajamos en linea colaborativamente.
